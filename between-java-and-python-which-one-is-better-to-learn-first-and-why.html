Between Java and Python, which one is better to learn first and why?

📅 Feb 23, 2020 11:40 PM PST  |  🌐 English

Hello World!

I would go with Python —

  1. It's the easiest language, without you needing to learn a lot of complex elements before you can write a hello world.
    1. Why do you need to know about data types, methods, objects classes to write a hello world?
    2. It doesn't scare beginners by heavyweight jargon and syntax on day 1
    3. Remember your kids learn cat, dog etc before they learn A, B, C
    4. once people understand programming they can learn complex world too.

Beyond Hello World

I would go with Python and/or javascript

  • python is a great choice for large scale application. You don't have to base your baise on 20 years old idiom of strongly typed language. Strong typing takes away dynamic design idea which is more productive and scalable.
  • Operator Overloading provides an elegance missing in Java. So is property. Decorators provide out of box AOP without 3rd party library.
  • Huge number of high performing library.

In my experience language like Java and c# are fighting battles for survival. A major share usurped by javascript and front end. Back end mostly needing data science and ML goes to Python.

This is based on my experience in industry over 20 years as consultant software architect and corporate trainer in almost every leading MNC.
