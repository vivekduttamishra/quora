Which of these is correct utterance OM or AUM?

📅 Apr 30, 2021 10:58 PM PDT  |  🌐 English

Honestly speaking neither is correct.

There is nothing that is equivalent to ॐ in English. It is unique in the sense that it is neither vowel nor consonant, it is both alphabet and a complete word, the simplest and and yet complex.

So technically neither is correct.

But if we need to ignore the limitations of the language we can use either of the spelling OM and AUM realizing that both are indicative only. But the pronunciation is A-U-M
