This question like many others keep popping up in different wordings. I had answered a similar question here and surprising my answer doesn't change a bit — Vivek Dutta Mishra (विवेक दत्त मिश्रा)'s answer to If Aryabhata discovered 0 (zero), then before that, how would we know that Shree Rama lived in the jungle for 14 years? How did they measure?

This is almost like asking — Newton discovered gravitation, how Apple used to fall on ground before this discovery.

To summarise —

  1. Things exists even before they are understood. Humans still had kidney and lever and they did their work before we studied them. So 100 people can exist before the knowledge of 0.
  2. Aryabhatt discovered 0, and if you study Gita you would know many knowledge gets lost over a period of time only to be rediscovered. 0 is one such thing.
  3. Roman number system doesn't have 0, and yet they can count thousands.
