How many arguments can be passed in the main function?

📅 May 06, 2020 02:16 AM PDT  |  🌐 English

Thanks for the A2A

It depends on the programming language.

But general principle any number of arguments can be passed that would be collected in an array. Often argument 0 may be program name itself.

c/c++

int main(int argc, char *argv[]) {…}

Java

class Whatever{
   public static void main(String [] args) {
... 
}

C#

class Whatever{
   public static void Main(String [] args) {
... 
}

Python

import sys

# argument accessible as sys.args
